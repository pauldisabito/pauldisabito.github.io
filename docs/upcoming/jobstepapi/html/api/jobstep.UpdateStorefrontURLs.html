Overview
Job Step UpdateStorefrontURLs
Updates the Storefront URLs for objects like categories, products, folders and content assets. This will update the URLs for all sites assigned under 'Scope'.
Execution Context:
Sites
Parallel Execution:
Not Supported
Always execute on restart:
Supported
Exit Status
OK
Step exit status used when updating Storefront URLs finished successfully.
ERROR
Step exit status used when updating Storefront URLs failed.
SKIPPED
Step exit status used when updating Storefront URLs has been skipped.
RESOURCES_UNAVAILABLE
Step exit status used when updating Storefront URLs has failed caused by unavailable resources.
