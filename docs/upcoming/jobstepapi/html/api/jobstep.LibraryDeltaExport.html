Overview
Job Step LibraryDeltaExport
Creates Delta Export for Libraries. Support must be contacted to enable delta exports.
Execution Context:
Organization
Parallel Execution:
Not Supported
Always execute on restart:
Supported
Input Parameters
Consumers  :  String (Required)
Comma Separated List of consumers. Limit of 1000 characters
ExportFile  :  String (Required)
Delta Export name and Export file name and path relative to 'IMPEX/src'.
LibraryIDs  :  String (Required)
Comma Separated list of Library Content Ids to be exported. Limit of 1000 characters
Exit Status
OK
Step exit status used when the export finished successfully.
ERROR
Step exit status used when the export failed.
