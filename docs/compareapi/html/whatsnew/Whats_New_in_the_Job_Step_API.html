What's New in the Job Step API

The following Job Steps are either new or have been changed in version 23.3.

ImportAssignments
  • The Job Step's Parameters have changed:
    ParameterSummary
    AfterImportFileHandling The allowed values changed from '[Archive, Archive Zipped, Keep, Delete]' to '[Keep, Delete, Archive, Archive Zipped, Archive In Subfolders, Archive Overwrite Existing]'.
    ImportMode The allowed values changed from '[Merge, Update, Replace, Delete]' to '[Merge, Delete, Replace, Update]'.
ImportCustomerGroups
  • The Job Step's documentation has changed.
ImportCustomers
  • The Job Step's documentation has changed.
ImportCustomObjects
  • The Job Step's Parameters have changed:
    ParameterSummary
    AfterImportFileHandling The allowed values changed from '[Archive, Archive Zipped, Keep, Delete]' to '[Keep, Delete, Archive, Archive Zipped, Archive In Subfolders, Archive Overwrite Existing]'.
    ImportMode The allowed values changed from '[Merge, Update, Replace, Delete]' to '[Merge, Delete, Replace, Update]'.
ImportStores
  • The Job Step's Parameters have changed:
    ParameterSummary
    AfterImportFileHandling The allowed values changed from '[Archive, Archive Zipped, Keep, Delete]' to '[Keep, Delete, Archive, Archive Zipped, Archive In Subfolders, Archive Overwrite Existing]'.
    ImportMode The allowed values changed from '[Merge, Update, Replace, Delete]' to '[Merge, Delete, Replace, Update]'.
