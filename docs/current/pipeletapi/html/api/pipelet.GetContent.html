Pipelet GetContent
This pipelet returns the Content from the current site's site library identified by the supplied ContentID. The pipelet will exit with an ERROR if any of the following conditions are encountered:
  • a ContentID is not supplied
  • content with the ContentID cannot be found
  • content is found but is not online
  • a content library for the site cannot be found
