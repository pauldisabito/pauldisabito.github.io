Overview
Job Step UndoPreconfiguredCodeReplicationProcess
Undo the preconfigured code replication process with the ID defined at parameter 'ReplicationProcessID'. The given code replication process must have been executed using a step of type 'ExecutePreconfiguredCodeReplicationProcess' by this job.
Execution Context:
Organization
Parallel Execution:
Not Supported
Always execute on restart:
Supported
Input Parameters
ReplicationProcessID  :  String (Required)
The ID of the preconfigured replication process.
Exit Status
ERROR
Used when the replication process could not be triggered or when the replication process failed.
OK
Used when the replication process completed successfully.
NO_STAGING_INSTANCE
Used when the step is used on non-staging instances.
