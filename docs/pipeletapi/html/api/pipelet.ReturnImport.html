Pipelet ReturnImport

This pipelet processes a return import file and results in returns being updated and / or created. It is used in a job.

The import process is tightly integrated with calls to java script custom hook implementations so allowing the import process to be highly customized. An example cartridge is available which demonstrates the customized import.
