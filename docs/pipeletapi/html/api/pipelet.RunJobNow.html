Pipelet RunJobNow
Run the job specified by JobName for the current organization. Only jobs that have been created via the Business Manager Scheduler may be run via this pipelet. This pipelet goes as far as queuing the job. The pipelet exits with ERROR if any of the following conditions are encountered: - The job does not exist. - The job is not enabled. - The job cannot be queued.
