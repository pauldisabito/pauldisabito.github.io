Pipelet SetRequestLocale
Sets the given locale for the current request. The error exit is taken, if given locale id is invalid, the locale is not active or the locale is not allowed for the site.
Group:
Util
Flags:
Error Connector
Input Parameters
LocaleID  :  String (Required)
The localeID to set at the request.
