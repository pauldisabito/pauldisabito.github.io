Overview
Job Step CatalogDeltaExport
Creates Delta Export for Catalogs. Support must be contacted to enable delta exports.
Execution Context:
Sites
Parallel Execution:
Not Supported
Always execute on restart:
Supported
Input Parameters
CatalogIDs  :  String (Required)
Comma Separated list of Catalog Ids to be exported. Character limit 1000
Consumers  :  String (Required)
Comma Separated List of consumers. Character limit 1000
ExportFile  :  String (Required)
Delta Export name and Export file name and path relative to 'IMPEX/src'.
MasterProductExport  :  Boolean (Optional)
If selected, the option exports all changes with the master product.
Default Value: false
Exit Status
OK
Step exit status used when the export finished successfully.
ERROR
Step exit status used when the export failed.
