Overview
Job Step ImportSiteArchive
Imports a site import archive file into the current instance.
Execution Context:
Organization
Parallel Execution:
Not Supported
Always execute on restart:
Supported
Input Parameters
ImportFile  :  String (Required)
Import file name located directly under 'IMPEX/src/instance' folder.
ImportFromRealm  :  Boolean (Optional)
If selected, the import file lookup will be taken in the global realm directory.
Default Value: false
ImportMode  :  String (Optional)
The import mode.
Allowed Values: merge
Default Value: merge
Exit Status
OK
Used when the import finished successfully.
ERROR
Used when the import failed.
